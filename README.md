# 射手座之日 原型版

这是一个浏览器运行的《射手座之日》即时舰队对战原型。

## 代码结构（已合并维护）

- `shared/game-core.js`：统一游戏规则核心（舰船运动、分离、技能、视野、AI、战斗结算）。
- `src/game.js`：本地 AI 模式前端，仅负责输入与渲染，规则由 `shared/game-core.js` 驱动。
- `server/server.js`：网络对战服务端，仅负责房间/同步/网络协议，规则同样由 `shared/game-core.js` 驱动。
- `src/online-client.js`：网络大厅与对战客户端渲染/插值/输入发送。

后续若要调整平衡或技能机制，优先修改 `shared/game-core.js`，本地与联机会同时生效。

- 网络对战大厅：`http://localhost:8080/online.html`

## 已实现机制

- 2D 实时对战，地图采用 3x3 九宫格战区。
- 开战前可在 7 名角色中选择 1 艘主舰与 2 艘副舰：
  - 凉宫春日、古泉一树、长门有希、朝比奈1096、阿虚、鹤屋学姐、朝仓凉子。
  - 每名角色拥有独立舰体、能量、航速与机动参数。
- 一级分离与二级分离：
  - 一级分离：副舰一独立。
  - 二级分离：副舰二独立。
  - 未分离时舰队转向更迟缓；分离后独立舰转向更灵活。
  - 任一编队的航速始终取该编队内最慢成员。
- 能量机制：
  - 低推进更容易积累能量。
  - 高推进会降低回能并可能出现净耗能。
  - 未分离时主舰队可用能量为队内总和；分离后各舰单独计算。
- 视野与侦察：
  - 仅显示当前视野内的敌方单位。
  - 敌方离开视野后不保留可见信息。
- 侦查机机制：
  - 只能指定派往某个战区。
  - 先直飞战区，再随机巡航。
  - 单次受击即毁，侦查机互撞同归于尽。
- 自动攻击机制：
  - 舰船与僚机自动攻击最近可见目标。
  - 具有弹道飞行与散布，可能落空。
  - 侧弦角度可获得更高输出效率。
- 旗舰 / 分舰技能：
  - 春日：`SOS团长` / `神说会赢的`
  - 古泉：`机关的力量` / `超能力`
  - 有希：`消失的世界` / `apm上万`
  - 1096：`过去与未来的我` / `1096光线`
  - 阿虚：`在虚构世界里寻求现实感的人才有问题` / `靠谱的普通人`
  - 鹤屋：`神秘赞助人` / `钞能力`
  - 朝仓：`我不会绕过你哦` / `刀锋女王`
- 敌方 AI：
  - 自动机动、分离、侦查、技能释放。
- 网络对战大厅：
  - 可选择创建公开房、私人房、AI训练房。
  - 私人房支持随机 6 位房间号加入。
  - 可加入公开房间进行玩家对战。
  - 采用服务器权威固定 Tick（30Hz）与快照同步（15Hz）。
  - 客户端使用 Ping/Jitter 自适应插值缓冲，降低高延迟抖动观感。
  - 输入序号确认（Ack）机制，减少输入乱序造成的状态误差。

## 角色数值与技能详表

以下数值与机制均按当前 `shared/game-core.js` 的实际实现整理。

### 通用规则

- 基础侦查机：
  - 消耗 `28` 编队能量。
  - 公共冷却 `2.6` 秒。
  - 进入指定战区后巡航，单次受击即毁。
- 射界火力密度：
  - 船头 `±60°`：`1x` 火力密度。
  - 侧舷 `±60° ~ ±120°`：`1.5x` 火力密度。
  - 后侧 `±120° ~ ±150°`：`1x` 火力密度。
  - 舰尾 `±150° ~ 180°`：`0x`，不自动开火。
  - 这里的倍率通过“开火频率”实现，不通过“单发伤害”实现。
- 分离与能量：
  - 未分离时，主舰编队技能消耗按编队总能量结算。
  - 分离后，副舰技能只消耗该副舰自身所在小编队的能量。
- 基础能量恢复：
  - 低推进更容易回能，高推进会压低回能。
  - 每名角色都有独立的基础回能值与移动耗能值。

### 凉宫春日

- 旗舰技能 `SOS团长`
  - 类型：主动。
  - 消耗 `68` 编队能量。
  - 冷却 `22` 秒。
  - 持续 `16` 秒。
  - 效果：对所有存活的分舰各随机附加 1 个 SOS Buff。
  - Buff 池：
    - `宇宙人`：视野 `x1.22`，射程 `x1.18`。
    - `超能力者`：转向 `x1.15`，伤害 `x1.16`。
    - `未来人`：航速 `x1.16`，回能 `x1.22`，加速度 `x1.14`。
    - `异世界人`：射速 `x1.18`，受伤倍率 `x0.82`。
- 分舰技能 `神说会赢的`
  - 类型：主动。
  - 消耗 `60` 能量。
  - 冷却 `20` 秒。
  - 持续 `8` 秒。
  - 效果：
    - 持续期间每次攻击有 `50%` 概率暴击，暴击造成 `3x` 单发伤害。
    - 持续期间会优先攻击自身射程与射界内最近的敌人，即使该目标暂未进入正常视野。
- 设计定位：
  - 春日本体面板偏高，技能以爆发和随机强化为主。
  - 适合当主舰做节奏核心，也适合分离后做高风险高收益副舰。

### 古泉一树

- 旗舰技能 `机关的力量`
  - 类型：主动。
  - 消耗 `58` 编队能量。
  - 冷却 `18` 秒。
  - 持续 `12` 秒。
  - 效果：
    - 全队加速度倍率 `x1.75`，持续 `12` 秒。
    - 施放后的前 `3` 秒，全队舰船免疫一切伤害。
- 分舰技能 `超能力`
  - 类型：可选落点主动技。
  - 消耗 `50` 能量。
  - 冷却 `22` 秒。
  - 最大闪现距离 `240`。
  - 效果：
    - 闪现到指定位置。
    - 若没有指定位置，则原地闪现并照常获得后续强化。
    - 下一次攻击的单发伤害提升为 `4x`，命中后强化清除。
- 设计定位：
  - 古泉旗舰负责强行开窗与团队冲刺。
  - 分舰形态则偏刺客式换位与重击。

### 长门有希

- 旗舰技能 `消失的世界`
  - 类型：被动。
  - 效果：
    - 封印本舰队全部角色技能。
    - 同时封印通用侦查机技能。
    - 我方三艘可选舰船各获得 `1` 次额外命数。
    - 触发复活时恢复到 `52%` 最大舰体，并至少回到 `40%` 能量。
- 分舰技能 `apm上万`
  - 类型：主动。
  - 消耗 `60` 能量。
  - 冷却 `24` 秒。
  - 效果：向 `8` 个方向各放出 `2` 架侦察机，总计 `16` 架。
  - 单架性能：
    - 飞行速度 `118`
    - 视野 `82`
    - 存活 `10.5` 秒
    - 在距离舰体约 `230` 的锚点附近短轨巡航
- 设计定位：
  - 有希旗舰是极端规则改写型角色。
  - 牺牲主动技能与通用技能，换来极高容错与稳定侦察覆盖。

### 朝比奈1096

- 旗舰技能 `过去与未来的我`
  - 类型：被动。
  - 效果：
    - 主舰位额外生成 1 艘 `1096僚舰`，编入主舰编队。
    - 主舰与僚舰都进入“半舰体模式”。
  - 当前实现数值：
    - 1096 基础舰体为 `640`。
    - 半舰体模式实际取 `0.55x`，即单舰最大舰体为 `352`。
    - 两舰合计为 `704` 总舰体，略高于单艘 1096，但更怕范围覆盖。
- 分舰技能 `1096光线`
  - 类型：主动指向技。
  - 消耗 `74` 能量。
  - 冷却 `12` 秒。
  - 蓄力 `1.05` 秒，命中特效持续 `0.26` 秒。
  - 射程 `1460`。
  - 命中判定半径 `11`。
  - 效果：对光线穿过的每艘敌舰造成其最大舰体 `28%` 的伤害。
  - 施放限制：
    - 只有分离后的 1096 副舰可用。
    - 若蓄力中被击毁或重新附着，光线会取消。
- 设计定位：
  - 旗舰形态提供额外船位密度。
  - 分舰形态则是中冷却的斩线压制工具。

### 阿虚

- 旗舰技能 `在虚构世界里寻求现实感的人才有问题`
  - 类型：被动。
  - 效果：
    - 全舰队转向性能显著改善。
    - 直接转向倍率额外 `x1.28`。
    - 加速度倍率额外 `x1.16`。
    - 航线约束中的最小转弯半径约降为 `0.62x`。
    - 初始可接受偏角约提升到 `1.4x`。
    - 全向火力密度近似统一，实际按 `1.02x` 处理。
- 分舰技能 `靠谱的普通人`
  - 类型：主动。
  - 消耗 `42` 能量。
  - 冷却 `18` 秒。
  - 持续 `14` 秒。
  - 效果：
    - 立即回复 `18%` 最大舰体。
    - 持续期间：
      - 转向 `x1.28`
      - 航速 `x1.08`
      - 伤害 `x1.08`
      - 加速度 `x1.12`
      - 受伤倍率 `x0.84`
- 设计定位：
  - 阿虚是“手感修正器”。
  - 当主舰时全队操作容错最高，当副舰时则是低成本自保强化。

### 鹤屋学姐

- 旗舰技能 `神秘赞助人`
  - 类型：主动。
  - 消耗 `60` 编队能量。
  - 冷却 `20` 秒。
  - 持续 `8` 秒。
  - 效果：
    - 全队技能冷却流逝速度 `x2`。
    - 全队所有舰船每秒恢复 `1%` 最大生命值。
- 分舰技能 `钞能力`
  - 类型：主动战区技。
  - 消耗 `66` 能量。
  - 冷却 `24` 秒。
  - 效果：夺取指定战区内的敌方侦察机与僚机。
  - 细则：
    - 只对 `侦察机` 和 `僚机` 生效。
    - 不会影响敌方主舰或副舰。
    - 被夺取单位会立刻改编到我方，并重设巡航目标到该战区中心。
    - 若目标战区没有可夺取单位，则技能失败，不进入冷却，并返还本次能量消耗。
- 设计定位：
  - 鹤屋旗舰适合在接敌前或对轰中期开启团队过载窗口。
  - 分舰技则是强战区干扰，用来反制侦察与前置僚机运营。

### 朝仓凉子

- 旗舰技能 `我不会绕过你哦`
  - 类型：主动。
  - 消耗 `64` 编队能量。
  - 冷却 `24` 秒。
  - 揭示持续 `2` 秒。
  - 效果：
    - 清除敌方全部由主动技能附加的增益。
    - 当前会清除的效果包括：
      - `SOS团长`
      - `神说会赢的`
      - `机关的力量`
      - 古泉分舰 `超能力` 的下一击强化
      - `靠谱的普通人`
      - `神秘赞助人`
      - `刀锋女王`
    - 同时强制揭示敌方全体单位位置 `2` 秒。
- 分舰技能 `刀锋女王`
  - 类型：主动。
  - 消耗 `52` 能量。
  - 冷却 `20` 秒。
  - 持续 `10` 秒。
  - 效果：
    - 航速 `x1.45`
    - 加速度 `x1.26`
    - 转向 `x1.12`
    - 持续期间与本舰接触的敌舰每秒受到其最大生命值 `2%` 的伤害。
- 设计定位：
  - 朝仓旗舰用于拆对手爆发窗口并强行开图。
  - 分舰则是高速贴身处刑单位，专门惩罚走位失误与混战贴脸。

## 操作说明

- 选择舰船：在侧栏“操作舰船”下拉框中切换主舰/副舰。
- 战区选择：左键单击地图，选择对应战区。
- 航行指令：左键双击地图设置目标点，自动生成可视化贝塞尔航线。
- 航线调整：拖拽黄色控制点调整弧度，拖拽绿色终点调整落点；系统会按当前航向、速度与转向能力自动限制为可行曲线。
- 推进档位：通过“推进功率”滑块独立控制，不再依赖拖拽手势。
- 清除航线：使用“清除当前舰船航线”按钮恢复直航。
- 战区行动：在侧栏选择战区后派出侦查机，或对带战区目标的技能施放。
- 指向型分舰技能：点击技能按钮后，再点击地图方向开火。
- 可选落点分舰技能：古泉的 `超能力` 先进入瞄准，点击地图可指定闪现点；再次点击技能按钮则原地释放。

## 网络大厅操作说明

- 连接服务器：页面加载后自动连接 `ws(s)://当前网页域名:21246`，无需手动输入地址；“重新连接”按钮可手动重连。
- 创建房间：
  - “创建公开房”会出现在公开房列表中。
  - “创建私人房”会显示 6 位房间号。
  - “创建 AI 训练房”可直接开局。
- 加入房间：
  - 在公开房列表点击“加入”。
  - 或输入 6 位房间号加入私人房。
- 对战控制：
  - 左键单击：选择战区（本地标记）。
  - 左键双击：为当前舰船设定目标点。
  - 拖拽控制点/终点：在线调整贝塞尔曲线。
  - 推进功率滑块独立控制，可实时同步至服务器。
