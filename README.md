# 射手座之日 原型版

这是一个浏览器运行的《射手座之日》即时舰队对战原型。

## 运行方式

在项目目录启动任意静态服务器即可：

```bash
cd /Users/haruhi/haruhi-game
python3 -m http.server 8080
```

浏览器打开 `http://localhost:8080`。

## 已实现机制

- 2D 实时对战，地图采用 3x3 九宫格战区。
- 主舰 + 两艘副舰的编队模式，共享编队血量逻辑。
- 一级分离与二级分离：
  - 一级分离：副舰一独立。
  - 二级分离：副舰二独立。
  - 每次分离后总属性约衰减到 80%（连续两次约为 64%）。
- 能量与速度联动：
  - 低推进更容易积累能量。
  - 高推进会降低回能并可能出现净耗能。
- 视野与侦察：
  - 仅显示当前视野内的敌方单位。
  - 敌方离开视野后不保留可见信息。
- 侦查机机制：
  - 只能指定派往某个战区。
  - 先直飞战区，再随机巡航。
  - 单次受击即毁，侦查机互撞同归于尽。
- 自动攻击机制：
  - 舰船与僚机自动攻击最近可见目标。
  - 具有弹道飞行与散布，可能落空。
  - 侧弦角度可获得更高输出效率。
- 舰长技能：
  - 凉宫春日（主舰）：向战区派遣战斗僚机。
  - 朝比奈实玖瑠（副舰二）：二级分离后可发射远程光束。
  - 长门有希（副舰一）：一级分离后视野扩大。
- 敌方 AI：
  - 自动机动、分离、侦查、技能释放。

## 操作说明

- 选择舰船：在侧栏“操作舰船”下拉框中切换主舰/副舰。
- 战区选择：左键单击地图，选择对应战区。
- 航行指令：左键双击地图设置目标点，自动生成可视化贝塞尔航线。
- 航线调整：拖拽黄色控制点调整弧度，拖拽绿色终点调整落点；系统会按当前航向、速度与转向能力自动限制为可行曲线。
- 推进档位：通过“推进功率”滑块独立控制，不再依赖拖拽手势。
- 清除航线：使用“清除当前舰船航线”按钮恢复直航。
- 战区行动：在侧栏选择战区后派出侦查机或僚机。
- 实玖瑠光束：点击技能按钮后，再点击地图方向开火。
